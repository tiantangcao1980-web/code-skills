# 用户输入 03 · 完整(11 项 + 视觉方向都齐)

项目: SnapMatte — 给开发者拍截图自动加留白与水印。
用户: 独立开发者 + 5-20 人小团队。
主流程: 1) 拖图进窗口  2) 自动渲染留白与水印  3) 一键保存到剪贴板。
平台: Web,响应式(桌面优先)。
技术栈: React + TS + Tailwind;后端不需要(纯前端 Canvas)。
不做: v0.1 不做登录、付费、协作、批量处理、移动端 App。
验收: 拖入 PNG,5 秒内生成结果图;移动端 Chrome / Safari 能跑;首屏 <2s。
部署: Vercel。
时间预算: 2 天。
外部依赖: 无第三方 API;Logo 我自己提供 SVG。
视觉方向: 参考 Vercel + Linear,深色 + 单色 + 一抹绿色,无炫光。
