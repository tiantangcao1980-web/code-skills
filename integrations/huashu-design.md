# huashu-design

- **URL**: https://github.com/alchaincyf/huashu-design
- **License**: 见原仓库
- **类型**: 文案 / 话术设计资源 / 哲学

## 这是什么

文案不是"翻译需求",文案是**人格体现** —— 每个产品要有统一的"嘴型"。huashu-design 把这件事方法化:语气坐标、关键文案规则、禁用词清单,以及大量"竞品对比"的反向案例。

## 在 code-skills 中的对应

- **对应 skill**:`copywriting-design`
- 借鉴的核心:
  - **语气四维度**(正式度/理性度/温度/幽默度)
  - **5 句关键文案规则**(主标题/CTA/空态/错态/成态)
  - **禁用词清单**(极致/亲/我们/操作成功 等)

## 何时用

- 产品起名 / slogan
- 首页主标题
- 按钮 / 表单 / 错误 / 空态文案审查
- 整理品牌语气坐标
- 把多人写的文案统一到同一"嘴型"

## 怎么用

直接触发本仓库 `copywriting-design` skill。例如:

> "帮我写首页主标题 + CTA,语气坐标:朋友 + 逻辑 + 克制 + 偶尔自嘲"

## 不集成的部分

- huashu-design 全部案例库(我们只取最高频的语气模型 + 5 句规则)
- 中文专属修辞细节(本仓库 skill 是中英通用的语气框架)
- 营销长文模板(本仓库聚焦产品文案,不是营销文案)
