# PRD 完整范例 · SnapMatte v0.1

> 11 章模板的实战版本(节选关键章)。

## 1. 一句话价值
**给开发者拍截图自动加优雅留白和品牌水印,5 秒搞定贴进文档**。

## 2. 背景与目标
- 来自 5 人深访(`docs/1-research-report.md`):
  - 5/5 提到截图发文档前要"修一下"才能发
  - 4/5 用过 Photoshop / Pixelmator / Canva,觉得"杀鸡用牛刀"
  - 3/5 提到"想要统一品牌风格"
- 目标:7 日留存 ≥ 40%(基于免费工具基线)

## 3. 核心场景(用户旅程)
| 阶段 | 发现 | 安装 | 首次使用 | 重复使用 | 推荐 |
|------|------|------|---------|----------|------|
| 行为 | HN/PH 上看到 | 直接打开 web | 拖图 | 工作流嵌入 | 在 PR 截图水印 |
| 情绪 | 🤔 | 😐 | 😀 | 😀 | 🥳 |
| 痛点 | _ | _ | 不知道怎么开始 | _ | _ |
| 机会 | _ | _ | 引导 1-2-3 | _ | 水印含小 url |

## 4. MVP 切片(v0.1)
- ✅ Must(5 条):
  1. 拖入 PNG → 5s 出图
  2. 默认主题(深色 + 留白 32px + 阴影)
  3. 一键复制剪贴板
  4. 移动端 Safari/Chrome 可用
  5. 首屏 <2s
- 🔵 v0.2 Should:自定义 Logo / 主色 / 多模板
- ⚪️ v0.3+ Could:批量、API、团队共享配置
- ❌ Won't:登录、付费、协作、原生 App

## 5. 详细需求(节选 1 条)
### 5.1 拖入出图
- JTBD: 截图发文档,想 5 秒贴进去
- 验收: Given 拖入 ≤ 5MB PNG, When 释放, Then 5 秒内出图 + 自动复制
- 设计: [Figma url 或 ASCII]
  ```
  ┌──────────────────────┐
  │   [大箭头图标]        │
  │   把截图拖到这里      │
  │   或 Cmd+V 粘贴       │
  └──────────────────────┘
  ```
- 状态: idle / dragging / processing / done / error
- 边界 case:
  - 文件 > 10MB → 提示压缩
  - 非 PNG/JPG → 提示格式
  - 处理失败 → 显示原图 + 错态文案
- 错态文案(引用 copywriting-design):"保存没成功,可能磁盘满了。再试一次?"

## 6. 接口契约
- 纯前端,无后端
- localStorage schema: `{theme, watermarkUrl, lastResult}`(详见 INTERFACE.md)

## 7. 成功指标
- **NSM**: 7 日留存(用户 D7 回来用)
- 输入指标(我们能动的):
  - 首页转化率(开页 → 拖图)
  - 拖图成功率
  - 拖图后复制率
- 输出指标(滞后但说明价值的):
  - 7 日留存
  - 推荐分享数
  - 月活账户

## 8. 非功能需求
- 性能: 首屏 < 2s, 拖图出图 < 5s
- 可用性: 99.5%(纯前端,只看 CDN)
- 兼容: Chrome 100+ / Safari 15+ / Firefox 100+,移动端 iOS 15+ / Android 10+
- 安全: 图片不上传服务器(纯前端 Canvas)

## 9. 风险登记册
| 风险 | 影响 | 概率 | 缓解 | 责任人 |
|------|------|------|------|--------|
| Canvas 处理大图卡死 | 用户流失 | 中 | 限 10MB + Worker 处理 | 前端 |
| 移动端 Safari Canvas Bug | 体验降级 | 中 | E2E 真机测 + 降级方案 | QA |
| 用户截图含敏感信息上传 | 隐私问题 | 低 | 明确"纯前端不上传"标注 | 产品 |
| AI 截图工具竞品出 | 用户流失 | 高 | 差异化定位 + 速度优势 | 产品 |
| 域名 / Vercel 服务挂 | 不可用 | 低 | 监控 + 备用域名 | 运维 |

## 10. 上线计划
- 灰度: 不需要(纯前端,无后端风险)
- 监控点: Vercel Analytics 5 个事件(open / drag / process_ok / process_fail / copy)
- 回滚条件: process_fail 率 > 5% → 回退上一版本

## 11. FAQ
- Q: 我的截图安全吗? A: 完全本地处理,不上传服务器
- Q: 支持团队 Logo 吗? A: v0.1 不支持,v0.2 加
- Q: 能批量吗? A: v0.1 不支持
- Q: 收费吗? A: v0.1 永久免费
- Q: 移动端能用吗? A: 可以,Safari/Chrome 都支持
